From a Helios Data class import and organize the asset.

Parameters:

  • cpg_asset (CPGAsset) – CPGAsset data class which contains information on how to import and construct the scene hierarchy.

  • with_skinning (bool) – If skinning should be kept or applied to the imported models.

  • skel_root_dict (dict{str|Joint}) – A dictionary of rig asset ids to the joints they represent. When importing with skinning if the rig asset id matches the incoming asset it will be bound against it instead of deleted. If skinning is enabled but a root dict was not provided this will be generated during the import process.

  • organize_only (bool) – If the scene should only build organization groups and not attempt to import.

Returns:

The imported organization group, and the skel_root_dict
