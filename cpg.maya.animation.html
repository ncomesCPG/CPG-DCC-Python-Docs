cpg.maya.animation.anim_curves.change_curve_start_time(animation_curve, start_difference)[source]

Updates the time range on an animation curve. Note: if there are more than one animation curve you are moving and one curve starts at frame 5 and another starts at frame 10. You want them all to move the same amount and you may need to change the start time to match.

Parameters:

  • animation_curve (pm.nt.AnimCurve) – The animation curve that the time will be updated.

  • start_difference (float) – How much the all frames should move.
